# CodeAlpha Secure Coding Review

## Task 3 - Secure Coding Review

### Project Overview
This project was completed as part of the CodeAlpha Cyber Security Internship. The objective was to review a deliberately vulnerable login application, identify security weaknesses, assess their impact, and recommend remediation measures.

## Vulnerabilities Identified
1. Hardcoded Credentials
2. Plain Text Password Storage
3. Username Enumeration
4. Sensitive Information Disclosure
5. Unlimited Login Attempts
6. Weak Password Policy
7. No Audit Logging
8. Poor Privilege Management
